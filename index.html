Information

This page is built using:

  • getUserMedia.
  • Web Audio API.
  • Voice-Change-O-Matic - MDN getUserMedia and Web Aduo API demo
  • Web GL API.
  • Three.js - A WebGL Framework.
  • KissFFT - A FFT implementation in C.
  • Emscripten - An LLVM-to-Javascript compiler. Used to compile KissFFT to Javascript. (I did not compile KissFFT by myself. I copied compiled js files from Forward real-complex FFT speed test)

Control:

  • Q,Z : Move Camera Forward and Backward
  • W,S,A,D : Rotate Camera
  • Arrows : Move Camera
  • Space : Toggle Pause
  • R : Reset Camera Position and Z Step
  • 1,2 : Increase/Decrease Z Step
  • M : [Line(Complex) style only] Sub style change
